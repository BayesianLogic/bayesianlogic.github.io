Download

BLOG can run on all operating systems that support Java. The minimal requirement is Java 1.6 or higher. BLOG interactive shell and debugger require scala 2.10.4 or above.

Stable Release

  • Universal zip
  • Linux debian/ubuntu pre-build package
  • Windows installation package

Tools

  • BLOG Pygment lexer A syntax highlight package for Latex and Markdown. (created by: Sharad Vikram and Lei Li)
  • BLOG syntax highlighting for sublime
  • BLOG syntax highlighting for VIM: blog.vim (created by Bharath Ramsundar)
  • BLOG syntax highlighting for Emacs (0.6): blog.el (created by: Wei Wang)
